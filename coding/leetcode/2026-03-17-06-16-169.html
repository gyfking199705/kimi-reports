LeetCode #169 - 多数元素

Boyer-Moore 投票算法 · 时间 O(n) 空间 O(1) 的最优解法

Easy 数组 投票算法 计数

📋 题目描述

给定一个大小为 n 的数组 nums，返回其中的多数元素。

多数元素是指在数组中出现次数 大于 ⌊ n/2 ⌋ 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

示例 1

输入：nums = [3,2,3]

输出：3

示例 2

输入：nums = [2,2,1,1,1,2,2]

输出：2

解释：2 出现 4 次，1 出现 3 次，2 是多数元素

💡 核心洞察

如果数组中存在一个元素出现次数超过 n/2，那么它的票数一定大于其他所有元素票数之和。

想象一场投票：每当两个不同的候选人「对决」时，他们互相抵消一票。最后剩下的，必定是得票过半的候选人。

📝 算法证明（为什么正确？）

核心思想：设多数元素为 maj，出现次数为 m > n/2。

  1. 每次遇到 maj，计数器 +1
  2. 每次遇到非 maj，计数器 -1
  3. 最坏情况下，maj 与其他所有元素「配对抵消」
  4. 剩余未抵消的 maj 数量 = m - (n-m) = 2m - n > 0
  5. 因此最后 count > 0，candidate 必为 maj
