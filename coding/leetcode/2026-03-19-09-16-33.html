LeetCode #33 - 搜索旋转排序数组

📋 题目描述

整数数组 nums 按升序排列，数组中的值 互不相同 。

在传递给函数之前，nums 在预先未知的某个下标 k（0 <= k < nums.length）上进行了旋转，使数组变为 [nums[k], nums[k+1], ..., nums[n-1], nums[0], nums[1], ..., nums[k-1]]。

给你旋转后的数组 nums 和一个整数 target，如果 nums 中存在这个目标值 target，则返回它的下标，否则返回 -1。

示例：

输入: nums = [4,5,6,7,0,1,2], target = 0
输出: 4

🧠 核心思路

关键洞察：旋转后的数组仍然部分有序——旋转点将数组分成两个有序区间。

通过比较中间元素与左端/右端元素，可以判断哪一半是有序的：

  • 若 nums[left] <= nums[mid]，则左半部分 [left, mid] 有序
  • 否则右半部分 [mid, right] 有序

确定有序区间后，只需判断 target 是否在该区间内，即可决定搜索方向。
