LeetCode #15 - 三数之和

3Sum — 双指针经典 / 去重技巧 / 面试高频

📋 题目描述

Medium 数组 双指针 排序

给你一个整数数组 nums，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k，同时还满足 nums[i] + nums[j] + nums[k] == 0。

请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

🎬 算法执行过程（点击步骤或代码行查看对应关系）

i 指针
left 指针
right 指针
三数之和为0

Python 实现（带详细注释）

📊 复杂度分析

指标 值 说明
时间复杂度 O(n²) 外层循环 O(n) + 双指针扫描 O(n)，最坏情况需要遍历所有组合
空间复杂度 O(1) / O(log n) ~ O(n) 忽略结果数组：排序原地 O(1)（Python Timsort O(n)）或 Java O(log n) 递归栈
关键优化 排序预处理 + 双指针 + 三处去重（nums[i], nums[left], nums[right]）

💡 核心要点总结

  • 排序是前提：只有有序后才能使用双指针技巧
  • 固定一个，双指针找另外两个：将 O(n³) 降为 O(n²)
  • 三处去重是关键：
    • i 去重：跳过相同的 nums[i]
    • left 去重：找到解后跳过相同的 nums[left]
    • right 去重：找到解后跳过相同的 nums[right]
  • 剪枝优化：当 nums[i] > 0 时可直接 break（后面的都大于0，不可能和为0）
