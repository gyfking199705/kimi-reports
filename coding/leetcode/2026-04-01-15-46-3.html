LeetCode #3 - 无重复字符的最长子串Medium

Longest Substring Without Repeating Characters | 滑动窗口经典模板题

📋 题目描述

给定一个字符串 s，请你找出其中不含有重复字符的最长子串的长度。

示例 1：
输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
示例 2：
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
示例 3：
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。

提示：0 <= s.length <= 5 * 10⁴，s 由英文字母、数字、符号和空格组成。

💡 核心思路

使用滑动窗口技巧：维护一个窗口 [left, right]，窗口内的字符保证不重复。右指针不断右移扩展窗口，遇到重复字符时左指针收缩窗口，直到窗口内无重复。过程中记录最大窗口长度。

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现



1class Solution:


2    def lengthOfLongestSubstring(self, s: str) -> int:


3        # 哈希集合，记录窗口内的字符


4        char_set = set()


5        left = 0  # 滑动窗口左边界


6        max_len = 0  # 记录最大长度


7        


8        # right 作为右边界遍历字符串


9        for right in range(len(s)):


10            # 如果当前字符已在窗口中，收缩左边界


11            while s[right] in char_set:


12                char_set.remove(s[left])


13                left += 1


14            


15            # 将当前字符加入窗口


16            char_set.add(s[right])


17            


18            # 更新最大长度


19            max_len = max(max_len, right - left + 1)


20        


21        return max_len

📊 复杂度分析

指标 值 说明
时间复杂度 O(n) 右指针遍历字符串一次，左指针最多也移动 n 次
空间复杂度 O(min(m, n)) 哈希集合大小，m 为字符集大小，n 为字符串长度

🔍 代码优化版（使用哈希表记录位置）



1class Solution:


2    def lengthOfLongestSubstring(self, s: str) -> int:


3        char_index = {}  # 记录字符最后一次出现的位置


4        left = 0


5        max_len = 0


6        


7        for right, char in enumerate(s):


8            # 如果字符出现过且在窗口内，直接跳到重复位置之后


9            if char in char_index and char_index[char] >= left:


10                left = char_index[char] + 1


11            


12            char_index[char] = right


13            max_len = max(max_len, right - left + 1)


14        


15        return max_len

优化版利用哈希表直接定位重复字符位置，避免逐个移除，但核心思想仍是滑动窗口。

📝 面试要点

  • 为什么用滑动窗口？ 因为题目要求「连续子串」，具有单调性，可以用双指针维护区间。
  • 如何证明是 O(n)？ 左右指针都只会向右移动，最多各移动 n 次。
  • 哈希集合 vs 哈希表？ 集合只需判断存在性，表可记录位置用于优化。
  • 变体应对： 如果问「最长子数组」而不是「子串」，思路相同，只是处理数组元素。
