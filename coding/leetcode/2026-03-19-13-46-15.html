LeetCode #15 - 三数之和

Medium 双指针 · 排序 · 去重技巧

📋 题目描述

给你一个整数数组 nums ，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k ，同时还满足 nums[i] + nums[j] + nums[k] == 0。

请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例：

输入: nums = [-1,0,1,2,-1,-4]

输出: [[-1,-1,2], [-1,0,1]]

🎯 核心解题思路

  • 🔢
    排序预处理

    先排序数组，使相同数字相邻，便于去重；同时有序数组支持双指针

  • 📌
    固定一个，双指针找另外两个

    外层遍历 i，内层用 left 和 right 指针在 i 右侧寻找满足 nums[i] + nums[left] + nums[right] = 0 的组合

  • ✂️
    三重去重

    i、left、right 三个位置都需要跳过重复元素，这是本题最容易出错的地方
