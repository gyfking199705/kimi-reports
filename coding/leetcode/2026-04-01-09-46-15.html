LeetCode #15 - 三数之和

📋 题目描述

给你一个整数数组 nums，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足：

  • i != j、i != k 且 j != k
  • nums[i] + nums[j] + nums[k] == 0

请你返回所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

示例 1：
输入：nums = [-1,0,1,2,-1,-4]
输出：[[-1,-1,2],[-1,0,1]]
解释：
- (-1) + 0 + 1 = 0 → [-1,0,1]
- (-1) + (-1) + 2 = 0 → [-1,-1,2]
示例 2：
输入：nums = [0,1,1]
输出：[]
解释：唯一可能的三元组和不为 0

📊 复杂度分析

维度复杂度说明
时间复杂度O(n²)外层循环O(n)，双指针内层O(n)，总计O(n²)
空间复杂度O(1) 或 O(log n)仅使用常数额外空间（不考虑输出和排序的栈空间）
💡 核心洞察：通过排序将O(n³)暴力解法优化到O(n²)，关键是固定一个数后，用双指针在有序数组中寻找两数之和。同时利用排序后数组的有序性，高效跳过重复元素。
