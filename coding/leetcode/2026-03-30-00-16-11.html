LeetCode #11 - 盛最多水的容器

Container With Most Water

Medium 双指针 贪心 数组

📋 题目描述

给定一个长度为 n 的整数数组 height。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i])。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

💡 注意：你不能倾斜容器。
面积计算公式：
area = min(height[left], height[right]) × (right - left)

🎬 算法执行过程（点击步骤或代码行查看对应关系）

示例数组: height = [1, 8, 6, 2, 5, 4, 8, 3, 7]

Python 实现

🧠 核心洞察：为什么移动较短的板？
因为面积由 短板决定。如果移动长板，宽度减小，而高度仍然受短板限制，面积必然减小。
只有移动短板，才有可能找到更高的板，从而弥补宽度减小的损失。

📊 复杂度分析

维度 值 说明
时间复杂度 O(n) 双指针从两端向中间移动，每个元素最多被访问一次
空间复杂度 O(1) 只使用了常数个变量（left, right, max_area）
