🔄 LeetCode #206 - 反转链表

📋 题目描述

难度：Easy

给你单链表的头节点 head，请你反转链表，并返回反转后的链表。

示例 1：

输入：head = [1,2,3,4,5]

输出：[5,4,3,2,1]

示例 2：

输入：head = [1,2]

输出：[2,1]

约束条件：

  • 链表中节点的数目范围是 [0, 5000]
  • -5000 <= Node.val <= 5000

Python 实现 - 迭代法


1# Definition for singly-linked list.
2class ListNode:
3    def __init__(self, val=0, next=None):
4        self.val = val
5        self.next = next
6
7class Solution:
8    def reverseList(self, head: ListNode) -> ListNode:
9        prev, curr = None, head
10        while curr:
11            next_temp = curr.next  # 暂存下一个节点
12            curr.next = prev       # 反转指针方向
13            prev = curr            # prev 前移
14            curr = next_temp       # curr 前移
15        return prev              # prev 成为新头节点

🧠 复杂度分析

指标 值 说明
时间复杂度 O(n) 只需遍历链表一次，n 为链表长度
空间复杂度 O(1) 只使用三个指针，原地反转

💡 核心要点

  • 三指针法：prev 跟踪已反转部分，curr 处理当前节点，next_temp 暂存下一个节点防止断链
  • 关键操作顺序：先保存 next → 反转指针 → 移动 prev → 移动 curr
  • 循环条件：curr 不为 None 时继续，即还有节点需要处理
  • 返回值：prev 最终指向原链表尾，即反转后的新头节点
  • 边界处理：空链表或单节点链表直接返回，循环不会执行，返回 prev=None 或正确处理

🔄 递归解法（拓展）

class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        # 递归终止条件：空链表或只有一个节点
        if not head or not head.next:
            return head
        
        # 递归反转后续链表
        new_head = self.reverseList(head.next)
        
        # 将当前节点与后续节点反转
        head.next.next = head
        head.next = None
        
        return new_head

递归解法思路：递归到链表末尾，然后逐层返回时将指针反向。时间复杂度 O(n)，空间复杂度 O(n)（递归栈空间）。
