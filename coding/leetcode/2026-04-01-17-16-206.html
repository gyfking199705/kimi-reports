🔄 LeetCode #206 - 反转链表

📋 题目描述

给你单链表的头节点 head，请你反转链表，并返回反转后的链表。


示例 1：

输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]

示例 2：

输入：head = [1,2]
输出：[2,1]

示例 3：

输入：head = []
输出：[]

链表 双指针 Easy 面试必会

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现 - 迭代法

📝 解题思路

💡 核心思想：遍历链表时，将当前节点的 next 指针改为指向前一个节点。需要三个指针：prev（前一个节点）、curr（当前节点）、next_temp（临时保存下一个节点）。

迭代法步骤：

  1. 初始化 prev = None（反转后链表的新尾）
  2. 遍历链表，curr 指向当前节点
  3. 保存 curr.next 到 next_temp（防止断链）
  4. 将 curr.next 指向 prev（核心反转操作）
  5. prev 和 curr 都向前移动一步
  6. 返回 prev（新的头节点）

📊 复杂度分析

方法 时间复杂度 空间复杂度 说明
迭代法 O(n) O(1) ⭐ 推荐，只需遍历一遍
递归法 O(n) O(n) 栈空间，面试可提
