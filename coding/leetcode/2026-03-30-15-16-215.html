LeetCode #215 - 数组中的第K个最大元素

Kth Largest Element in an Array

📋 题目描述

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

示例

输入: nums = [3,2,1,5,6,4], k = 2
输出: 5

输入: nums = [3,2,3,1,2,4,5,5,6], k = 4
输出: 4
Medium 快速选择 分治 随机化算法 堆

🎯 核心思路：Quick Select 算法

核心洞察：快排的思想，但只递归处理包含目标的那一半区间。平均时间复杂度 O(n)，空间 O(log n)。

为什么不用排序？ 完全排序需要 O(n log n)，但我们只需要第 k 大，不需要知道其他元素的具体顺序。

Quick Select 的关键：

  • 随机选择 pivot（避免最坏情况）
  • partition 后，pivot 的位置就是它在有序数组中的最终位置
  • 如果 pivot 位置恰好是第 k 大，直接返回
  • 否则只在包含第 k 大的那一半继续递归

📊 复杂度分析

方法 时间复杂度 空间复杂度 说明
完全排序 O(n log n) O(1) / O(log n) 最简单但不够最优
最小堆 O(n log k) O(k) k 很小时效果好
Quick Select ⭐ O(n) 平均 / O(n²) 最坏 O(log n) 随机化后实际接近线性
