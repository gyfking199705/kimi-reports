LeetCode #2 - 两数相加

Medium 链表 | 递归 | 模拟

📋 题目描述

给你两个 非空 的链表，表示两个非负的整数。它们每位数字都是按照 逆序 的方式存储的，并且每个节点只能存储 一位 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

示例 1：
输入：l1 = [2,4,3], l2 = [5,6,4]
输出：[7,0,8]
解释：342 + 465 = 807
示例 2：
输入：l1 = [0], l2 = [0]
输出：[0]
示例 3：
输入：l1 = [9,9,9,9,9,9,9], l2 = [9,9,9,9]
输出：[8,9,9,9,0,0,0,1]

🎬 算法执行过程（点击步骤或代码行查看对应关系）

Python 实现



1# Definition for singly-linked list.


2class ListNode:


3    def __init__(self, val=0, next=None):


4        self.val = val


5        self.next = next


6

7class Solution:


8    def addTwoNumbers(self, l1: ListNode, l2: ListNode) -> ListNode:


9        # 哑节点作为结果链表的头节点前驱


10        dummy = ListNode(0)


11        current = dummy


12        carry = 0  # 进位


13        


14        # 当还有节点或进位时继续循环


15        while l1 or l2 or carry:


16            # 获取当前位的值，节点为空则取0


17            val1 = l1.val if l1 else 0


18            val2 = l2.val if l2 else 0


19            


20            # 计算当前位的和及新的进位


21            total = val1 + val2 + carry


22            carry = total // 10


23            digit = total % 10


24            


25            # 创建新节点并连接


26            current.next = ListNode(digit)


27            current = current.next


28            


29            # 移动到下一个节点


30            if l1: l1 = l1.next


31            if l2: l2 = l2.next


32        


33        return dummy.next
💡 核心思路
模拟小学竖式加法：从个位（链表头）开始逐位相加，记录进位。
关键点：使用哑节点(dummy)简化头节点处理，循环条件包含进位处理边界情况。

📊 复杂度分析

度量 复杂度 说明
时间复杂度 O(max(M, N)) M、N分别为两个链表长度，遍历较长链表一次
空间复杂度 O(max(M, N)) 结果链表长度最多为 max(M, N) + 1

🔑 解题技巧

  • 哑节点技巧：使用 dummy 节点避免对头节点的特殊处理，最后返回 dummy.next
  • 循环条件设计：while l1 or l2 or carry 确保所有位和最终进位都被处理
  • 空节点处理：用 l1.val if l1 else 0 优雅处理不等长链表
  • 整数溢出：本题不能先转换成整数再相加，因为链表可能极长导致溢出
