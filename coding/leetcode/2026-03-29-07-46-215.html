LeetCode #215 - 数组中的第K个最大元素

📋 题目描述

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

要求：设计并实现时间复杂度为 O(n) 的算法。

难度: Medium 分治 快速选择 随机化

示例: nums = [3,2,1,5,6,4], k = 2 → 输出: 5

🎬 算法执行过程（点击步骤或代码行查看对应关系）

执行步骤

步骤1: 计算目标索引
target = len(nums) - k，找升序第 target 小的元素
步骤2: 随机选择 pivot
避免最坏情况，随机选pivot并与末尾交换
步骤3: Partition 分区
将小于pivot的元素移到左边，大于的移到右边
步骤4: 扫描数组
遍历元素，小于pivot就与store_index位置交换
步骤5: 放置 pivot
将pivot放到store_index位置，此时它处于正确排序位置
步骤6: 判断与递归
比较pivot位置与target，决定返回或递归哪一半

Python 实现



1import random


2


3class Solution:


4    def findKthLargest(self, nums, k):


5        target = len(nums) - k


6        return self.quickSelect(nums, 0, len(nums)-1, target)


7    


8    def quickSelect(self, nums, left, right, target):


9        pivot_idx = random.randint(left, right)


10        nums[pivot_idx], nums[right] = nums[right], nums[pivot_idx]


11        


12        store = left


13        for i in range(left, right):


14            if nums[i] < nums[right]:


15                nums[store], nums[i] = nums[i], nums[store]


16                store += 1


17        


18        nums[store], nums[right] = nums[right], nums[store]


19        


20        if store == target: return nums[store]


21        elif store < target:


22            return self.quickSelect(nums, store+1, right, target)


23        else:


24            return self.quickSelect(nums, left, store-1, target)

📊 复杂度分析

指标 值 说明
时间复杂度 O(n) 平均 / O(n²) 最坏 每次只处理一半数组，T(n) = T(n/2) + O(n) = O(n)
空间复杂度 O(log n) 递归栈深度，平均情况
稳定性 不稳定 partition交换会改变元素相对顺序

💡 核心考点

  • 快速选择算法：基于快速排序的partition思想，但只递归处理包含目标的那一半
  • 随机化优化：随机选择pivot避免最坏情况（如已排序数组）
  • 第k大 ↔ 第(n-k)小：转换为找升序后索引为n-k的元素
  • 分治策略：根据pivot位置决定搜索左半还是右半
