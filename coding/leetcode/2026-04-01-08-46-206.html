LeetCode #206 - 反转链表

Reverse Linked List | Easy | 链表基础必会题

📋 题目描述

给你单链表的头节点 head，请你反转链表，并返回反转后的链表。

示例：

输入：head = [1,2,3,4,5]
输出：[5,4,3,2,1]
输入：head = [1,2]
输出：[2,1]

进阶：可以迭代或递归地反转链表。你能否用两种方法解决这道题？

🐍 Python 实现（迭代法）

💡 核心思路

📌 关键洞察：三指针法

  • prev：指向已反转部分的头节点（初始为 null）
  • curr：当前正在处理的节点
  • next_temp：暂存下一个节点，防止断链

链表反转的本质是改变每个节点的 next 指针方向。我们需要三个指针：

  1. 遍历链表，每次取出当前节点 curr
  2. 先将 curr.next 保存到 next_temp（防止丢失后续链表）
  3. 将 curr.next 指向 prev（反转操作）
  4. prev 和 curr 依次向前移动

📊 复杂度分析

方法 时间复杂度 空间复杂度 说明
迭代法 O(n) O(1) 只需常数空间，最优解法 ⭐
递归法 O(n) O(n) 递归栈空间，代码更简洁
