LeetCode #3 - 无重复字符的最长子串

滑动窗口 哈希表 双指针 Medium

📋 题目描述

给定一个字符串 s，请你找出其中不含有重复字符的最长子串的长度。

示例

示例 1：

输入: s = "abcabcbb"

输出: 3

解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2：

输入: s = "bbbbb"

输出: 1

解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3：

输入: s = "pwwkew"

输出: 3

解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。

约束条件

  • 0 <= s.length <= 5 × 10⁴
  • s 由英文字母、数字、符号和空格组成

💡 核心思路

1
滑动窗口思想

维护一个窗口 [left, right]，窗口内的字符保证不重复。right 指针向右扩展窗口，遇到重复字符时移动 left 指针收缩窗口。

2
哈希表优化

使用字典记录每个字符最后一次出现的位置，O(1) 时间判断字符是否在窗口内，避免遍历查找。

3
双指针技巧

left 和 right 两个指针都只会向右移动，每个字符最多被访问两次，保证 O(n) 时间复杂度。

📊 复杂度分析

维度 复杂度 说明
时间复杂度 O(n) right 指针遍历字符串一次，left 指针最多移动 n 次，每个字符最多被访问两次
空间复杂度 O(min(m, n)) m 为字符集大小，实际存储的是窗口内字符，最多 min(m, n) 个
最优性 ✓ 最优 至少需要遍历每个字符一次，O(n) 已达理论下界

🔍 关键点解析

为什么 left = max(left, char_index[s[right]] + 1)？

当发现重复字符时，需要将 left 移动到重复字符位置的下一个位置。但是，如果重复字符在 left 的左边（不在当前窗口内），则 left 不应该回退，所以取 max。

示例：s = "abba"

  • right=0: 'a'，窗口="a"，max_len=1
  • right=1: 'b'，窗口="ab"，max_len=2
  • right=2: 'b' 重复！left = max(0, 1+1) = 2，窗口="b"（第二个b）
  • right=3: 'a'，char_index['a']=0，left = max(2, 0+1) = 2，窗口="ba"

注意：如果不取 max，left 会回退到 1，导致错误！
