📋 题目描述

给定一个只包括 '('，')'，'{'，'}'，'['，']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合
  • 左括号必须以正确的顺序闭合
  • 每个右括号都有一个对应的相同类型的左括号

💡 核心考点

栈的后进先出(LIFO)特性完美匹配括号的嵌套结构。遇到左括号入栈，遇到右括号检查栈顶是否匹配。这是栈数据结构最经典的应用场景。

🔍 完整解法

class Solution:
    def isValid(self, s: str) -> bool:
        # 用字典存储括号配对关系
        mapping = {')': '(', ']': '[', '}': '{'}
        stack = []
        
        for char in s:
            if char in mapping:
                # 右括号：检查栈顶是否匹配
                top = stack.pop() if stack else '#'
                if top != mapping[char]:
                    return False
            else:
                # 左括号：入栈
                stack.append(char)
        
        # 栈为空说明全部匹配成功
        return not stack

⏱️ 复杂度分析

指标值说明
时间复杂度O(n)只遍历字符串一次，每个字符最多入栈出栈一次
空间复杂度O(n)最坏情况下所有字符都是左括号，栈大小为n

🎓 解题思路详解

1. 为什么用栈？

括号具有对称性和嵌套性。最内层的括号必须先闭合，这正是栈「后进先出」的特性。

2. 算法步骤

  1. 初始化空栈和括号映射表
  2. 遍历字符串中的每个字符
    • 遇到左括号：压入栈中
    • 遇到右括号：弹出栈顶，检查是否匹配
  3. 遍历结束后，栈为空则有效，否则无效

3. 边界情况

  • 字符串为空：直接返回 True（空字符串视为有效）
  • 右括号比左括号多：栈为空时遇到右括号，匹配失败
  • 左括号比右括号多：遍历结束栈不为空，匹配失败
  • 类型不匹配：(] 这样的组合要识别出来
