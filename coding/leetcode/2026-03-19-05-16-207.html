📋 题目描述

你这个学期必须选修 numCourses 门课程，记为 0 到 numCourses - 1。

在选修某些课程之前需要一些先修课程。先修课程按数组 prerequisites 给出，其中 prerequisites[i] = [a, b]，表示如果要学习课程 a 则 必须先 学习课程 b。

请判断是否可能完成所有课程的学习？如果可以，返回 true；否则，返回 false。

核心问题：判断有向图中是否存在环 —— 如果存在环（如 A→B→C→A），则无法完成所有课程。

示例：

  • numCourses = 2, prerequisites = [[1,0]] → true（先学0再学1）
  • numCourses = 2, prerequisites = [[1,0],[0,1]] → false（循环依赖）

🧠 算法思路：Kahn算法（BFS拓扑排序）

拓扑排序用于解决有向图的先后顺序问题。Kahn算法基于入度（有多少课程依赖于它）的思想：

  1. 计算每个课程的入度（有多少前置课程）
  2. 将入度为0的课程加入队列（没有前置依赖，可以直接学）
  3. 依次学习这些课程，并将它们的后置课程的入度减1
  4. 如果某后置课程入度变为0，说明前置都学完了，可以入队
  5. 最后统计能学完的课程数，等于总数则无环

💡 关键点总结

  • 入度数组：记录每门课程的前置依赖数量
  • 邻接表：记录每门课程的后置课程（学完它能解锁哪些课）
  • 队列：维护当前可以学习的课程（入度为0）
  • 计数器：统计实际学习的课程数量，用于最后判断
易错点：构建图时，[a, b] 表示 b→a（先学b才能学a），不要把方向搞反！
