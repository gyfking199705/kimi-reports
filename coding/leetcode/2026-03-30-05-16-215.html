LeetCode #215 - 数组中的第K个最大元素

Kth Largest Element in an Array

📋 题目描述

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。

难度: Medium 数组 分治 快速选择 堆(优先队列)
示例 1：
输入: nums = [3,2,1,5,6,4], k = 2
输出: 5

示例 2：
输入: nums = [3,2,3,1,2,4,5,5,6], k = 4
输出: 4

💡 核心思路

快速选择算法 (Quick Select) —— 快速排序的变种，但只递归处理包含目标的那一半区间。

  • 使用快速排序的分区思想，选定一个基准值(pivot)
  • 将数组分成两部分：大于pivot的放右边，小于pivot的放左边
  • 比较pivot的最终位置与目标索引 n-k
  • 只递归处理包含目标的那一半，平均时间复杂度 O(n)

为什么不用排序？排序需要 O(n log n)。快速选择利用「我们不需要完全有序，只需要知道第k大在哪」的特点，达到线性平均时间。

Python 实现 - 快速选择

import random

class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        def quickSelect(left, right, k_smallest):
            if left == right:
                return nums[left]
            
            pivot_index = random.randint(left, right)
            pivot_index = partition(left, right, pivot_index)
            
            if k_smallest == pivot_index:
                return nums[k_smallest]
            elif k_smallest < pivot_index:
                return quickSelect(left, pivot_index - 1, k_smallest)
            else:
                return quickSelect(pivot_index + 1, right, k_smallest)
        
        def partition(left, right, pivot_index):
            pivot = nums[pivot_index]
            nums[pivot_index], nums[right] = nums[right], nums[pivot_index]
            store_index = left
            
            for i in range(left, right):
                if nums[i] < pivot:
                    nums[store_index], nums[i] = nums[i], nums[store_index]
                    store_index += 1
            
            nums[right], nums[store_index] = nums[store_index], nums[right]
            return store_index
        
        return quickSelect(0, len(nums) - 1, len(nums) - k)

📊 复杂度分析

方案 时间复杂度 空间复杂度 说明
排序 O(n log n) O(log n) ~ O(n) 最简单，但不满足题目要求
快速选择 ⭐ O(n) 平均 / O(n²) 最坏 O(1) 随机pivot避免最坏情况
最小堆 O(n log k) O(k) k较小时很高效

🐍 最小堆解法（备选）

import heapq

class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        # 维护一个大小为k的最小堆
        # 堆顶就是第k大的元素
        heap = []
        for num in nums:
            heapq.heappush(heap, num)
            if len(heap) > k:
                heapq.heappop(heap)  # 弹出最小值
        return heap[0]  # 堆顶即第k大

当 k 远小于 n 时，堆解法也很有竞争力。例如 k=10, n=10^6 时，只需要 O(n log k) ≈ O(20n)。

🎯 面试考点

  • 为什么选随机pivot？ 避免已排序数组导致O(n²)最坏情况
  • 能否达到严格O(n)？ 可以，用「中位数的中位数」选pivot，但常数很大，面试通常不要求
  • 快速选择 vs 堆？ 快速选择空间更优O(1)，堆更稳定不怕最坏情况
