📦 LeetCode #11 - 盛最多水的容器

Container With Most Water — 双指针算法的经典入门题

Medium 双指针 贪心算法 数组

📋 题目描述

给定一个长度为 n 的整数数组 height。有 n 条垂线，第 i 条线的两个端点是 (i, 0) 和 (i, height[i])。

找出其中的两条线，使得它们与 x 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

说明：你不能倾斜容器。

示例

输入: height = [1,8,6,2,5,4,8,3,7]
输出: 49
解释: 选择第 1 条线(高度8)和第 8 条线(高度7)
      容器宽度 = 8 - 1 = 7
      容器高度 = min(8, 7) = 7
      面积 = 7 × 7 = 49

约束条件

  • n == height.length
  • 2 <= n <= 105
  • 0 <= height[i] <= 104

💡 解题思路

核心洞察：容器的面积 = 宽度 × 最小高度。双指针从两端开始，每次移动高度较小的那个指针——因为移动较高的指针不可能获得更大的面积（宽度在减小，高度受限于较小的那个）。

为什么移动较小的指针？

假设左指针在 i，右指针在 j，且 height[i] < height[j]：

  • 当前面积 = (j - i) × height[i]
  • 如果移动右指针 j → 宽度减小，高度最多还是 height[i]，面积必然减小
  • 如果移动左指针 i → 虽然宽度也减小，但新的高度 可能 更大，面积 可能 增大

因此，我们唯一有希望找到更大面积的做法是：移动高度较小的指针。

📝 代码详解

1class Solution:
2    def maxArea(self, height: List[int]) -> int:
3        # 初始化双指针和最大面积
4        left, right = 0, len(height) - 1
5        max_area = 0
6        
7        # 当两个指针相遇时结束
8        while left < right:
9            # 计算当前容器的面积
10            # 宽度 = 右指针 - 左指针
11            # 高度 = 两边高度的较小值
12            width = right - left
13            h = min(height[left], height[right])
14            current_area = width * h
15            
16            # 更新最大面积
17            max_area = max(max_area, current_area)
18            
19            # 移动高度较小的指针
20            # 关键：移动较高的指针不可能得到更大面积
21            if height[left] < height[right]:
22                left += 1
23            else:
24                right -= 1
25        
26        return max_area
