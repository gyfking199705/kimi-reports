LeetCode #20 - 有效的括号

Valid Parentheses — 栈的经典应用，面试必考基础题

📋 题目描述

给定一个只包括 '('、')'、'{'、'}'、'['、']' 的字符串 s，判断字符串是否有效。

有效字符串需满足：

  • 左括号必须用相同类型的右括号闭合
  • 左括号必须以正确的顺序闭合
  • 每个右括号都有一个对应的相同类型的左括号

示例：

输入: s = "([{}])"
输出: true

输入: s = "([)]"
输出: false

输入: s = "(("
输出: false

💡 解题思路

核心思想：栈 (Stack)

遍历字符串时：

  • 遇到左括号 (、{、[ → 将其对应的右括号压入栈
  • 遇到右括号 )、}、] → 与栈顶元素比较，不匹配则无效
  • 最后栈必须为空，说明所有括号都成功匹配

💡 为什么压入右括号？这样可以直接比较，无需额外的映射表转换

Python 实现


1class Solution:
2    def isValid(self, s: str) -> bool:
3        # 用字典存储括号匹配关系
4        mapping = {')': '(', '}': '{', ']': '['}
5        stack = []  # 模拟栈结构
6        
7        for char in s:
8            if char in mapping:  # 遇到右括号
9                # 栈为空 或 栈顶不匹配 → 无效
10                if not stack or stack[-1] != mapping[char]:
11                    return False
12                stack.pop()  # 匹配成功，弹出
13            else:  # 遇到左括号
14                stack.append(char)  # 压入栈
15        
16        return len(stack) == 0  # 栈空则全部匹配

🎯 核心考点总结

  • 栈 (Stack) 的基本操作：push、pop、peek，后进先出 (LIFO) 特性
  • 括号匹配的本质：最近未匹配的左括号，必须由最近的右括号闭合
  • 边界情况：空字符串(有效)、单个左/右括号(无效)、嵌套多层括号
