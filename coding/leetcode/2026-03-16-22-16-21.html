📎 LeetCode #21 - 合并两个有序链表

Easy 链表 · 双指针 · 递归

📋 题目描述

将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

示例 1：

输入：list1 = [1,2,4], list2 = [1,3,4]

输出：[1,1,2,3,4,4]

示例 2：

输入：list1 = [], list2 = []

输出：[]

示例 3：

输入：list1 = [], list2 = [0]

输出：[0]

💡 核心思路：使用虚拟头节点（dummy）简化边界处理，用双指针遍历两个链表，每次取较小节点接入新链表。

🎬 算法执行过程

点击左侧步骤查看对应代码高亮，或点击右侧代码行查看执行说明

步骤1: 初始化虚拟头节点
创建dummy节点作为新链表起点，避免处理空链表边界
dummy → None
步骤2: 创建当前指针
curr指向新链表的最后一个节点，用于拼接
curr = dummy
步骤3: 循环比较节点值
当两个链表都未结束时，比较当前节点的值
list1.val vs list2.val
步骤4: list1节点更小
将list1当前节点接入新链表
curr.next = list1
步骤5: list1指针后移
list1指向下一个待处理节点
list1 = list1.next
步骤6: list2节点更小
将list2当前节点接入新链表
curr.next = list2
步骤7: list2指针后移
list2指向下一个待处理节点
list2 = list2.next
步骤8: curr指针后移
curr始终指向新链表最后一个节点
curr = curr.next
步骤9: 连接剩余链表
其中一个链表已遍历完，直接连接另一个剩余部分
curr.next = list1 or list2
步骤10: 返回结果
dummy.next即为合并后的链表头节点
return dummy.next

🐍 Python 实现

 1 # Definition for singly-linked list.
 2 # class ListNode:
 3 #     def __init__(self, val=0, next=None):
 4 #         self.val = val
 5 #         self.next = next
 6 
 7 class Solution:
 8     def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
 9         # 虚拟头节点，简化边界处理
10         dummy = ListNode(-1)
11         curr = dummy
12         
13         # 双指针遍历两个链表
14         while list1 and list2:
15             if list1.val <= list2.val:
16                 curr.next = list1
17                 list1 = list1.next
18             else:
19                 curr.next = list2
20                 list2 = list2.next
21             curr = curr.next
22         
23         # 连接剩余部分
24         curr.next = list1 if list1 else list2
25         
26         return dummy.next

📝 复杂度分析

复杂度类型 值 说明
时间复杂度 O(n + m) 需要遍历两个链表的所有节点各一次
空间复杂度 O(1) 只使用了常数额外空间（几个指针）

注意：递归解法空间复杂度为 O(n+m)（递归栈空间），迭代解法更优。

🔁 递归解法（补充）

class Solution:
    def mergeTwoLists(self, list1: Optional[ListNode], list2: Optional[ListNode]) -> Optional[ListNode]:
        # 边界条件
        if not list1:
            return list2
        if not list2:
            return list1
        
        # 选择较小节点作为头，递归合并剩余部分
        if list1.val <= list2.val:
            list1.next = self.mergeTwoLists(list1.next, list2)
            return list1
        else:
            list2.next = self.mergeTwoLists(list1, list2.next)
            return list2

💡 面试要点

  • 虚拟头节点技巧：处理链表问题时，dummy节点能极大简化边界判断，是链表题的标准起手式
  • 指针操作：注意 curr.next 赋值和 curr = curr.next 的区别，前者修改链表结构，后者移动指针
  • 边界处理：当输入链表为空时，dummy.next 仍为 None，无需额外判断
  • 递归理解：递归解法更优雅但空间复杂度更高，理解递归版本有助于理解函数式编程思想
