LeetCode #15 - 三数之和

难度: Medium | 标签: 双指针排序数组

📋 题目描述

给你一个整数数组 nums，判断是否存在三元组 [nums[i], nums[j], nums[k]] 满足 i != j、i != k 且 j != k，同时还满足 nums[i] + nums[j] + nums[k] == 0。

请你返回所有和为 0 且不重复的三元组。注意：答案中不可以包含重复的三元组。

示例：
输入: nums = [-1,0,1,2,-1,-4]
输出: [[-1,-1,2], [-1,0,1]]

输入: nums = [0,1,1]
输出: []

输入: nums = [0,0,0]
输出: [[0,0,0]]

💡 核心思路

排序 + 双指针是解决此题的最优方案：

  1. 首先对数组进行排序，这样可以将重复元素聚集在一起，方便去重
  2. 固定第一个数 nums[i]，问题转化为在 i+1 到末尾的区间中寻找两数之和等于 -nums[i]
  3. 使用双指针在有序数组中高效查找两数之和
  4. 通过跳过重复元素来实现答案去重
关键去重技巧：
• 外层循环跳过重复的 nums[i]
• 找到答案后，left 和 right 都要跳过重复值
• 这样才能保证三元组不会重复
