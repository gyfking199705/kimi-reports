🔥 LeetCode #739 - 每日温度

📋 题目描述

给定一个整数数组 temperatures ，表示每天的温度，返回一个数组 answer ，其中 answer[i] 是指对于第 i 天，下一个更高温度出现在几天后。如果气温在这之后都不会升高，请在该位置用 0 来代替。


示例：

输入: temperatures = [73, 74, 75, 71, 69, 72, 76, 73]
输出: [1, 1, 4, 2, 1, 1, 0, 0]

📊 复杂度分析

指标 复杂度 说明
时间复杂度 O(n) 每个元素最多入栈出栈一次
空间复杂度 O(n) 栈最多存储n个索引（最坏情况：递减数组）

💡 核心思路

  • 单调递减栈：栈中保存的是索引，对应的温度值从栈底到栈顶递减
  • 关键观察：当遇到比栈顶温度高的新温度时，栈顶元素找到了它的「下一个更大元素」
  • 计算答案：answer[栈顶索引] = 当前索引 - 栈顶索引
  • 栈的作用：暂存那些还没找到「下一个更大温度」的天数索引
  • 为什么递减：只有递减才能保证栈顶是距离当前天最近的、还没找到答案的天
