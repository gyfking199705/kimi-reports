LeetCode #42 - 接雨水

Trapping Rain Water · 双指针 · 单调栈 · 动态规划

Hard 双指针 单调栈 动态规划 数组

📋 题目描述

给定 n 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

💡 核心洞察

每个位置能接的雨水 = min(左侧最高, 右侧最高) - 当前高度

如果当前位置是两边最高中的较低者，且高于当前高度，就能接到雨水。

示例：

输入: height = [0,1,0,2,1,0,1,3,2,1,2,1]
输出: 6
解释: 上面是由数组 [0,1,0,2,1,0,1,3,2,1,2,1] 表示的高度图，在这种情况下，
      可以接 6 个单位的雨水（蓝色部分表示雨水）

解法 时间复杂度 空间复杂度 特点
暴力解法 O(n²) O(1) 对每个位置遍历找左右最高
动态规划 O(n) O(n) 预处理左右最高数组
双指针法 ⭐ O(n) O(1) 最优空间，实时维护左右最高
单调栈 O(n) O(n) 按行计算，可扩展性强

🧠 三种解法详解

1. 动态规划法

预处理两个数组：left_max[i] 表示位置 i 左侧最高柱子，right_max[i] 表示右侧最高柱子。

def trap_dp(self, height: List[int]) -> int:
    if not height: return 0
    
    n = len(height)
    left_max = [0] * n
    right_max = [0] * n
    
    # 从左向右扫描左侧最高
    left_max[0] = height[0]
    for i in range(1, n):
        left_max[i] = max(left_max[i-1], height[i])
    
    # 从右向左扫描右侧最高
    right_max[n-1] = height[n-1]
    for i in range(n-2, -1, -1):
        right_max[i] = max(right_max[i+1], height[i])
    
    # 计算总雨水量
    ans = 0
    for i in range(n):
        ans += min(left_max[i], right_max[i]) - height[i]
    return ans

2. 双指针法 ⭐ 推荐

关键优化

当 left_max < right_max 时，位置 left 的雨水量只与 left_max 有关。
反之亦然。这样可以在一次遍历中实时计算，无需额外数组。

3. 单调栈法

按"行"计算雨水。维护一个单调递减栈，当遇到更高的柱子时，可以形成凹槽接水。

def trap_stack(self, height: List[int]) -> int:
    stack = []  # 存储索引，对应高度单调递减
    ans = 0
    
    for i, h in enumerate(height):
        # 当前柱子高于栈顶，可以接水
        while stack and h > height[stack[-1]]:
            top = stack.pop()
            if not stack:
                break
            # 计算宽度
            width = i - stack[-1] - 1
            # 计算高度（取两边较小者减去当前）
            bounded_height = min(h, height[stack[-1]]) - height[top]
            ans += width * bounded_height
        stack.append(i)
    return ans
