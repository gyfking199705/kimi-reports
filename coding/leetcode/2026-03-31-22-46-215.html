🎯 LeetCode #215 - 数组中的第K个最大元素

Medium 经典算法 快排 堆

📋 题目描述

给定整数数组 nums 和整数 k，请返回数组中第 k 个最大的元素。

请注意，你需要找的是数组排序后的第 k 个最大的元素，而不是第 k 个不同的元素。

你必须设计并实现时间复杂度为 O(n) 的算法解决此问题。

💡 关键理解：
第k大 = 升序排列后的倒数第k个 = 下标为 len(nums) - k 的元素
例如：nums=[3,2,1,5,6,4], k=2 → 排序后[1,2,3,4,5,6] → 第2大是5

🔧 解法对比

核心思想：基于快速排序的分区思想，每次分区后判断第k大元素在左半部分还是右半部分，只递归处理一半数据

Python 实现（快速选择）
核心思想：维护一个大小为k的最小堆，堆顶就是第k大的元素。遍历数组，如果当前元素比堆顶大就替换 
import heapq

class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        # 维护大小为k的最小堆
        min_heap = []
        
        for num in nums:
            if len(min_heap) < k:
                heapq.heappush(min_heap, num)
            elif num > min_heap[0]:
                heapq.heapreplace(min_heap, num)
        
        return min_heap[0]
核心思想：直接排序后取倒数第k个元素。虽然不符合题目O(n)的要求，但面试时可先提这个解法 
class Solution:
    def findKthLargest(self, nums: List[int], k: int) -> int:
        nums.sort()
        return nums[-k]

📊 复杂度分析

解法 时间复杂度 空间复杂度 适用场景
快速选择 O(n) 平均 / O(n²) 最坏 O(log n) 递归栈 追求最优时间，数据随机

🎯 面试要点

  • 快排分区思想：partition函数返回pivot的最终位置
  • 下标转换：第k大 = 第(n-k)小（升序）
  • 最坏情况：数组已有序，每次只减少一个元素 → O(n²)
  • 优化：随机选择pivot避免最坏情况
  • 空间优化：原地修改数组，只需O(log n)递归栈
