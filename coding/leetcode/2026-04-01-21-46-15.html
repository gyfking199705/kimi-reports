LeetCode #15 - 三数之和

Medium 排序 双指针 数组

📋 题目描述

给你一个包含 n 个整数的数组 nums，判断 nums 中是否存在三个元素 a, b, c，使得 a + b + c = 0？

请你找出所有和为 0 且不重复的三元组。

注意：答案中不可以包含重复的三元组。

💡 示例：
输入：nums = [-1, 0, 1, 2, -1, -4]
输出：[[-1, -1, 2], [-1, 0, 1]]

解释：
- nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0
- nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0
- nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0
⚠️ 关键点：去重是本题的核心难点！需要同时避免三元组整体重复，也要避免固定元素和双指针移动时的重复。

🎯 算法核心思想

排序 + 双指针是解决本题的最优策略：

  1. 排序：将数组排序，方便后续去重和双指针移动
  2. 固定第一个数：遍历数组，固定 nums[i]
  3. 双指针找另外两个数：在 i 之后的区间用双指针找和为 -nums[i] 的两个数
  4. 去重：在固定和移动指针时都要跳过重复元素

💡 解题技巧总结

  • 排序是前提：排序后才能用双指针，才能方便去重
  • 剪枝优化：当 nums[i] > 0 时，后面不可能有三数之和为 0
  • 三处去重：固定 i 时、left 移动时、right 移动时都要去重
  • 双指针移动：和太大则 right--，和太小则 left++
