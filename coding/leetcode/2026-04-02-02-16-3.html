LeetCode #3 - 无重复字符的最长子串

📋 题目描述

给定一个字符串 s，请你找出其中不含有重复字符的 最长子串 的长度。


示例 1:

输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。

示例 2:

输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

示例 3:

输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
注意：答案必须是子串的长度，"pwke" 是一个子序列，不是子串。

📊 复杂度分析

复杂度类型 数值 说明
时间复杂度 O(n) 每个字符最多被访问两次（右指针扩展一次，左指针收缩一次）
空间复杂度 O(min(m, n)) 哈希集合大小取决于字符集大小 m 和字符串长度 n 的最小值

💡 解题思路

核心思想：滑动窗口 (Sliding Window)


使用双指针维护一个"窗口"，窗口内的字符串保证无重复字符：

  1. 右指针扩展：不断向右移动右指针，将新字符加入窗口
  2. 遇到重复：如果新字符已在窗口中，移动左指针收缩窗口，直到窗口内无重复
  3. 记录最大值：每次扩展后，用当前窗口长度更新最大长度

使用哈希集合 set 在 O(1) 时间内判断字符是否重复。
