LeetCode #206 - 反转链表

Easy  |  链表  |  双指针

📋 题目描述

给你单链表的头节点 head，请你反转链表，并返回反转后的链表。

链表的定义：每个节点包含一个值 val 和指向下一个节点的指针 next。

🎯 核心考点

链表操作基本功——指针重定向。反转的本质是让每个节点的 next 指向它的前一个节点，而不是后一个节点。

示例

输入：head = [1, 2, 3, 4, 5]

输出：[5, 4, 3, 2, 1]

Python 实现（迭代法）



1class ListNode:


2    def __init__(self, val=0, next=None):


3        self.val = val


4        self.next = next


5


6class Solution:


7    def reverseList(self, head: ListNode) -> ListNode:


8        prev = None           # 前一个节点（已反转部分）


9        curr = head           # 当前要处理的节点


10        


11        while curr:           # 还有节点待处理


12            next_node = curr.next   # 保存下一个节点


13            curr.next = prev        # 反转：当前节点指向prev


14            prev = curr             # prev前移


15            curr = next_node        # curr前移


16        


17        return prev           # prev就是新头节点

📝 递归法实现（拓展）



1class Solution:


2    def reverseList(self, head: ListNode) -> ListNode:


3        # 终止条件：空节点或只剩一个节点


4        if not head or not head.next:


5            return head


6        


7        # 递归反转后面的链表


8        new_head = self.reverseList(head.next)


9        


10        # 反转当前节点与下一个节点的关系


11        head.next.next = head   # 下一个节点指向当前节点


12        head.next = None         # 当前节点变为尾节点


13        


14        return new_head

💡 关键思路总结

  • 三指针法：prev（已反转）、curr（当前处理）、next_node（临时保存）
  • 核心操作：curr.next = prev —— 让当前节点指向之前的节点
  • 顺序很重要：必须先保存next，再修改curr.next，否则断链
  • 终止条件：当curr为None时，prev就是新链表的头
