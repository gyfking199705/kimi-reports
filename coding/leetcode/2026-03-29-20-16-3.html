LeetCode #3 - 无重复字符的最长子串

滑动窗口的经典入门题 | 中等难度

📋 题目描述

给定一个字符串 s ，请你找出其中不含有重复字符的 最长子串的长度。

示例 1：
输入: s = "abcabcbb"
输出: 3
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。
示例 2：
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。

🎬 算法执行可视化（点击步骤或代码行查看对应关系）

Python 实现（滑动窗口）

📊 复杂度分析

维度复杂度说明
时间复杂度O(n)每个字符最多被访问两次（右指针进入窗口，左指针移出窗口）
空间复杂度O(min(m, n))m 是字符集大小，需要哈希集合存储窗口内字符

💡 核心思路

滑动窗口是一种在数组/字符串上维护一个动态区间的技巧：

  • 用 left 和 right 两个指针表示窗口的左右边界
  • 右指针不断向右移动扩大窗口，将新字符纳入考虑
  • 当发现窗口内有重复字符时，移动左指针缩小窗口，直到无重复
  • 每次右指针移动后，更新最大长度
关键优化：使用哈希集合（HashSet）在 O(1) 时间内判断字符是否在窗口内，避免 O(n) 的遍历检查。
