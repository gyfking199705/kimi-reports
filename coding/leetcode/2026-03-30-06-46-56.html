LeetCode #56 - 合并区间

区间问题经典入门 | 排序 + 贪心思想

📋 题目描述

以数组 intervals 表示若干个区间的集合，其中单个区间为 intervals[i] = [starti, endi]。

请你合并所有重叠的区间，并返回一个不重叠的区间数组，该数组需恰好覆盖输入中的所有区间。

示例 1：
输入：intervals = [[1,3],[2,6],[8,10],[15,18]]
输出：[[1,6],[8,10],[15,18]]
解释：区间 [1,3] 和 [2,6] 重叠, 将它们合并为 [1,6]。
示例 2：
输入：intervals = [[1,4],[4,5]]
输出：[[1,5]]
解释：区间 [1,4] 和 [4,5] 可被视为重叠区间（因为是闭区间）。

约束条件：

  • 1 ≤ intervals.length ≤ 104
  • intervals[i].length == 2
  • 0 ≤ starti ≤ endi ≤ 104
数组 排序 贪心

💡 核心思路

关键洞察：如果我们将所有区间按起始时间排序，那么重叠的区间必然相邻出现。

排序后，从左到右遍历区间，维护一个当前正在合并的区间：

  1. 如果新区间的起始时间 ≤ 当前区间的结束时间，说明有重叠，扩展当前区间的结束时间
  2. 如果新区间的起始时间 > 当前区间的结束时间，说明无重叠，保存当前区间，开启新区间

这种贪心策略确保我们总是尽可能合并更大的区间。

📝 总结

合并区间是区间问题的入门经典，核心技巧是：

  • 排序预处理——让重叠区间相邻，为贪心创造条件
  • 贪心策略——能合并就合并，不能合并就开新区间
  • 闭区间注意——本题是闭区间，相等也算重叠

掌握这道题后，可以顺利解决会议室系列、无重叠区间等进阶题目。
