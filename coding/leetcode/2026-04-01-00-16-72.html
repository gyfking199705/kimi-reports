LeetCode #72 - 编辑距离 Hard

动态规划 字符串 经典难题

📋 题目描述

给你两个单词 word1 和 word2，请返回将 word1 转换成 word2 所使用的最少操作数。

你可以对一个单词进行如下三种操作：

  • 插入 一个字符
  • 删除 一个字符
  • 替换 一个字符

示例 1

输入：word1 = "horse", word2 = "ros"

输出：3

解释：

horse → rorse (将 'h' 替换为 'r')

rorse → rose (删除 'r')

rose → ros (删除 'e')

示例 2

输入：word1 = "intention", word2 = "execution"

输出：5

💡 核心思路

这是一个经典的动态规划问题。定义 dp[i][j] 表示将 word1[0..i-1] 转换成 word2[0..j-1] 所需的最少操作数。

状态转移方程：

如果 word1[i-1] == word2[j-1]:
  dp[i][j] = dp[i-1][j-1]   // 字符相同，无需操作

否则:
  dp[i][j] = min(
    dp[i-1][j] + 1,   // 删除 word1[i-1]
    dp[i][j-1] + 1,   // 在 word1 中插入 word2[j-1]
    dp[i-1][j-1] + 1   // 替换 word1[i-1] 为 word2[j-1]
  )

关键是理解：增删改三种操作实际上是对称的。word1的删除 = word2的插入，word1的插入 = word2的删除。

📊 复杂度分析

指标 值 说明
时间复杂度 O(m × n) 需要填充 m×n 的DP表格，m和n分别是两个单词长度
空间复杂度 O(m × n) 二维DP数组空间，可优化至 O(min(m,n))
最优解法 DP + 滚动数组优化 空间优化后 O(min(m,n))

🚀 空间优化版本

1class Solution:
2    def minDistance(self, word1: str, word2: str) -> int:
3        m, n = len(word1), len(word2)
4        # 确保 word1 是较短的，节省空间
5        if m < n:
6            return self.minDistance(word2, word1)
7        
8        # 只保留两行：上一行和当前行
9        prev = list(range(n + 1))
10        curr = [0] * (n + 1)
11        
12        for i in range(1, m + 1):
13            curr[0] = i  # 第一列：word1[0..i-1] 转空串需 i 次删除
14            for j in range(1, n + 1):
15                if word1[i - 1] == word2[j - 1]:
16                    curr[j] = prev[j - 1]
17                else:
18                    curr[j] = min(
19                        prev[j] + 1,      # 删除
20                        curr[j - 1] + 1,  # 插入
21                        prev[j - 1] + 1   # 替换
22                    )
23            prev, curr = curr, prev  # 交换引用，复用数组
24        
25        return prev[n]
